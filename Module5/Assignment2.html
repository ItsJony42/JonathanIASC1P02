Skylanders

Skylanders is a historic game series of so many people's
childhood, including my own. There are 6 games total in the
series.


Spyro's Adventure
Giants
Swap Force
Trap Team
Superchargers
Imaginators

Skylanders Spyro's Adventure

The first game of the series is Skylanders Spyro's Adventure.
This game introduces all the core features and mechanics, as well
as the main cast and the main villain. The main cast of skylanders
includes;

  • Spyro
  • Stealth Elf
  • Gill Grunt
  • Trigger Happy
  • Eruptor
  • Terrafin
  • Hex

Skylanders Giants

The second game is Skylanders Giants. This game introduces
a new type of character, being clunky giant creatures that
can do unique tasks which normal sklanders can't. These 8
giants are;

  • Tree Rex
  • Crusher
  • Hot Head
  • Bouncer
  • Eye-Brawl
  • Thumpback
  • Swarm
  • Ninjini

Skylanders Swap Force

The third game is Skylanders Swap Force. This game introduces
another new type of character, which is a skylander with a top
and bottom half, which can be exchanged with other skylanders of
the same type, allowing for interesting combinations. In my
opinion, this game is the best of the series, as it has the
best story, levels, characters, and game mechanics.

Skylanders Trap Team

The fourth game is Skylanders Trap Team. This game doesn't introduce
any "new" character, just some skylanders that unlock a new feature.
What this game introduces are villains, which are bosses that can be
caught and used, allowing for new game mechanics that make the game
unique and fun. What this game does best are the boss fights, which
are the best in the series. If I had to rank my top 10, the list would
be;

  1. Dr. Krankcase
  2. Dream Catcher
  3. Golden Queen
  4. Wolfgang
  5. Chompy Mage
  6. Chef Pepper Jack
  7. Nightshade
  8. Luminous
  9. Gulper
  10. Kaos

Skylanders Superchargers

The fifth game is Skylanders Superchargers, and this game
is my second favourite of the series. It introduces a
racing mechanic into the game, which is by far the most
unique feature in the series (as of that time).













Skylanders Imaginators

The final game of the series is Skylanders
Imaginators, and this one lets players create their own
skylanders with their own powers and weapons, easily
the coolest feature ever.
